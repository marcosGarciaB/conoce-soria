    1- Instalar Node.js.
    2- Instalar Expo CLI: 
        -> npm install -g expo-cli
    3- Crear el proyecto con TypeScript.
        -> npx create-expo-app front-mobile --template expo-template-blank-typescript
    4- Navegar al directorio del proyecto.

        -> npm start
